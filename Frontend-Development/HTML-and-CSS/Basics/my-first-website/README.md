## **HTML Basic Structure Project**

### **Task Instructions**
1.  **Project Setup:** Create a folder named `my_first_website`.
2.  **File Creation:** Inside that folder, create three files:
    * `index.html`
    * `about.html`
    * `style.css` (Leave this file empty for now).
3.  **Linking:** Ensure the CSS file is linked in the `<head>` of both HTML pages.
4.  **Navigation:** Create a menu at the top to navigate between your two pages and external links (Google/Facebook).
5.  **Page Anchors:** Create "In-page" links that jump to specific sections on the same page. Create up to five Sections
6. **Image:** Add a local image and any external image image to the project


**Question:** "Look at the code in your `index.html` file. Explain the difference between the link created for **'About Us'** and the link created for **'Section 1'**. Specifically:
1. What attribute is used to make these links work?
2. Why does one link take you to a new file while the other simply moves the screen up or down on the same page? 
3. If you move the `style.css` file into a new folder named `assets`, how would you change the code in the `<head>` to keep the link working?"
