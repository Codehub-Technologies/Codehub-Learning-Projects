# Frontend Design System Guide

###  Image Aspect Ratios
#### Common Ratios:
- 1:1 (Square)
- 4:3 (Standard)
- 16:9 (Widescreen)
#### Usage:
- Profile Pictures: 1:1
- Thumbnails: 4:3
- Banners/Headers: 16:9

### Grid System
- Container Width: 1200px
- Columns: 12-column grid
- Gutter Width: 24px
- Margin: Auto to center content
